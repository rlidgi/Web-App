# Setup Instructions:
# 
# 1. For Gmail:
#    - Enable 2-factor authentication on your Google account
#    - Generate an App Password: https://myaccount.google.com/apppasswords
#    - Use your email and the app password (not your regular password)
#
# 2. For Outlook:
#    - Use your regular email and password
#    - You may need to enable "Less secure app access"
#
# 3. For SendGrid (recommended for production):
#    - Sign up at https://sendgrid.com
#    - Create an API key
#    - Use "apikey" as the username and your API key as the password

# Security Notes:
# - Never commit your .env file to version control
# - Use app passwords or API keys instead of regular passwords
# - Consider using Azure Key Vault for production environments
